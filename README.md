# E-Commerce-Product-Gallery
INTERN ID - CITS4013
The E-Commerce Product Gallery is a responsive frontend web application developed using HTML, CSS, and JavaScript. It displays products in an attractive grid layout, allowing users to browse items, search by product name, and filter products by category. The application provides an interactive shopping experience with product images, prices, and an Add to Cart button. It features a clean, modern user interface that adapts to different screen sizes, making it suitable for both desktop and mobile devices.

Key Features
Responsive product gallery
Product search functionality
Category-based filtering
Product images and pricing
Add to Cart functionality
Modern and user-friendly interface
Mobile-friendly responsive design
